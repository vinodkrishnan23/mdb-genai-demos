# mdb-genai-demos

### SA's gen-ai toolkit for ready to use demos
- mql_bot - Use mongodb vectorsearch capability to convert natural language to aggregation pipeline - Semantic search & RAG
- travelmate - Chat with a mongodb collection to plan your vacations - Semantic search & RAG
- researchmate - Leverage state of art Voyage AI - multi-modal embedding model to build RAG solution eliminating traditional complex need for PDF chunking.
